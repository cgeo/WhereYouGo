WhereYouGo is an unofficial client for Wherigo Geocaching. It uses offline vector maps, alternatively Locus can be used to display maps. Cartridges for the game can be downloaded at
www.wherigo.com

Download cartridges for Pocket PC Device. Other versions are not tested (but it's possible they will work). Download cartridges into 'sdcard/WhereYouGo' directory, or you can specify your own directory in settings.

If any wherigo cache causes trouble and you would like to fix that, feel free to open a bug report on our bugtracker.

Happy caching and nice weather!
