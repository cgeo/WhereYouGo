WhereYouGo ist ein inoffizieller Client für Wherigo-Geocaching.

Es unterstützt das Spielen von Wherigo-Cartridges, ein interaktives standortbasiertes Geocaching-Spiel. Du kannst ein Wherigo-Spiel laden, spielen und den aktuellen Stand speichern. Ein Kompass und Online- oder Offline-Karten können zur Navigation verwendet werden, alternativ kann Locus zu Kartenanzeige verwendet werden.
Cartridges zur Nutzung mit WhereYouGo können von https://www.wherigo.com und anderen Diensten heruntergeladen werden.

Hauptfunktionen:
- Automatisches Herunterladen von Cartridges von wherigo.com (kann ausgelöst werden von deinem Browser durch Öffnen einer Listing-Seite auf wherigo.com oder von anderen Geocaching-Apps wie c:geo)
- Benutze den Kompass um zur nächsten Spielzone oder Position zu navigieren
- Spielzonen und Standorte auf der Karte anzeigen
- Verwendet Online-Karten aus verschiedenen Quellen
- Unterstützt Offline-Kartendateien (derzeit nur im Mapsforge v0.3 Format)  
- Nahtlose Zusammenarbeit mit Locus um Karten anzuzeigen (alternativ zu den internen Karten)
- Mehrere Speicherplätze pro Cartridge
- Automatisches Speichern beim Wechsel zu einer anderen App
- Integrierter QR-Code Reader

Über WhereYouGo:

WhereYouGo ist OpenSource und lizensiert unter der GNU General Public License V3.0.

Nachdem der Quellcode für mehrere Jahre nicht mehr gewartet wurde, war nun ein neues Entwicklungsteam (das c:geo Team) mit Zustimmung des ursprünglichen Autors in der Lage den Quellcode zu übernehmen und die Entwicklung der App fortzusetzen.
Wenn du uns helfen kannst und willst, findest du Informationen und den Quellcode unter dem unten stehenden Link. Wir brauchen deine Hilfe: https://github.com/cgeo/WhereYouGo
Der Kern der Anwendung basiert auf OpenWIG, sie nutzt die Mapsforge-Bibliothek um Karten anzuzeigen, alternativ kann Locus verwendet werden.

Kommentare:
- When manually downloading cartridges from wherigo.com, make sure to use the version "Pocket PC Device". 
- Kopiere die Cartridges in deinen Wherigo-Ordner (Einstellungen - Hauptmenü - Wherigo-Ordner) Make sure you granted the application permission to write to external storage and you have write permission in selected directory. Some newer devices don't allow writing to SD card. If you're not sure, set "Wherigo folder" automatically.
- If you have problems installing or using WhereYouGo please contact us at whereryougo@cgeo.org 
- If you would like to know why WhereYouGo needs the permissions requested, please check https://github.com/cgeo/WhereYouGo/blob/master/PRIVACY.md for an explanation.
