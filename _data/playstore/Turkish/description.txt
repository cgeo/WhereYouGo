WhereYouGo, Wherigo Geoaching için resmi olmayan bir istemcisidir.

Etkileşimli konum tabanlı bir Geocaching oyunu olan Wherigo kartuşlarının kullanılmasına yarar. Wherigo oyununuzun son halini kaydedebilir, yükleyebilir veya oynayabilirsiniz. Compass and online or offline maps can be used to navigate, alternatively Locus can be used to display maps.
Cartridges for usage with WhereYouGo can be downloaded from https://www.wherigo.com and other services.

Ana özellikler:
- Automatic download of cartridges from wherigo.com (can be triggered from your browser by opening a listing page on wherigo.com or from other geocaching apps like c:geo)
- Sıradaki oyun bölgesine veya konuma yönlendirmek için pusulayı kullanır
- Haritada oyun bölgelerini ve konumları görüntüler
- Değişik kaynaklardan çevrimiçi haritaları kullanır
- Çevrimdışı harita dosyalarını destekler (Şimdilik sadece Mapsforge v0.3 formatı)  
- Seamless interworking with Locus to display maps (alternatively to internal maps)
- Kartuş başına birçok oyun kayıt yuvası bulundurur
- Başka bir uygulamaya geçildiğinde otomatik oyun kaydedilebilir
-Dahili Kare Kod okuyucu bulundurur

After the source code was unmaintained for several years, a new development team (the c:geo team) was - with agreement of the orignal author - finally able to take over the source code and continue app development.
Eğer bize yardım etmek isterseniz, aşağıdaki bağlantıdan gerekli bilgiye ve kaynak koduna ulaşabilirsiniz. Yardımınıza ihtiyacımız var: https://github.com/cgeo/WhereYouGo
The core of the application is based on OpenWIG, it uses the Mapsforge library to display maps, alternatively Locus can be used.

Not:
- When manually downloading cartridges from wherigo.com, make sure to use the version "Pocket PC Device". 
- Kartuşları Wherigo dosyanıza kopyalayın (Ayarlar - Başlıca - Wherigo dosyası). Make sure you granted the application permission to write to external storage and you have write permission in selected directory. Bazı yeni cihazlarda SD kart üzerine veri kaydına izin verilmez. Emin değilseniz, "Wherigo dosyası"nı otomatik oluşturun.
- If you have problems installing or using WhereYouGo please contact us at whereyougo@cgeo.org 
- If you would like to know why WhereYouGo needs the permissions requested, please check https://github.com/cgeo/WhereYouGo/blob/master/PRIVACY.md for an explanation.
