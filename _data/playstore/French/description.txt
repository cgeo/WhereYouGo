WhereYouGo est un client non officiel pour Wherigo Geoaching.

Il prend en charge les cartouches Wherigo, un jeu de géocaching interactif basé sur la localisation. Vous pouvez charger, jouer et enregistrer l'état de votre partie Wherigo. La boussole ainsi que des cartes en ligne ou hors ligne peuvent être utilisées pour naviguer, ou bien on peut utiliser Locus pour afficher des cartes.
Les cartouches à utiliser avec WhereYouGo peuvent être téléchargées à partir de https://www.wherigo.com et d'autres services.

Fonctionnalités principales :
- Téléchargement automatique des cartouches depuis wherigo.com (peut être déclenché à partir de votre navigateur en ouvrant une page descriptive sur wherigo.com ou à partir d'autres applications de géocaching comme c:geo)
- Utilisez la boussole pour accéder à la zone de jeu ou à la position suivante
- Afficher les zones de jeu et les positions sur une carte
- Utilisez des cartes en ligne venant de diverses sources
- Accepte les fichiers de carte hors ligne (uniquement le format Mapsforge v0.3 pour le moment)  
- Interaction transparente avec Locus pour afficher des cartes (en remplacement des cartes internes)
- Plusieurs emplacements de sauvegardes de jeu par cartouche
- Sauvegarde automatique du jeu lors du passage à une autre application
- Lecteur de code QR intégré

Après plusieurs années sans maintenance du code source, une nouvelle équipe de développement (l'équipe c:geo) a finalement été capable - avec l'accord de l'auteur original - de reprendre le code source et de poursuivre le développement de l'application.
Si vous êtes capable et désireux de nous aider, vous pouvez trouver des informations et le code source grâce au lien ci-dessous. Nous avons besoin de votre aide sur https://github.com/cgeo/WhereYouGo
Le cœur de l'application est basé sur OpenWIG, elle utilise la bibliothèque Mapsforge pour afficher des cartes, Locus peut être employé alternativement.

Notes :
- Lorsque vous téléchargez les cartouches sur le site wherigo.com, assurez-vous de choisir la version "Pocket PC Device" (appareil Pocket PC). 
- Copiez les cartouches dans votre dossier Wherigo (Paramètres - Global - Dossier Wherigo). Assurez-vous que vous avez autorisé l'application à écrire sur le stockage externe et que vous avez la permission d'écriture dans le répertoire sélectionné. Certains appareils parmi les plus récents n'autorisent pas l'écriture sur la carte SD. Si vous n'êtes pas sûr(e), sélectionnez le "Dossier Wherigo" par défaut.
- Si vous rencontrez des problèmes d'installation ou d'utilisation de WhereYouGo, veuillez nous contacter à whereyougo@cgeo.org 
- Si vous souhaitez savoir pourquoi WhereYouGo a besoin des autorisations demandées, veuillez consulter https://github.com/cgeo/WhereYouGo/blob/master/PRIVACY.md pour une explication.
